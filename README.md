# InstaInfo

The easy way to get IG user id from username & vice versa

## Prerequisites

* Node  
* Axios  
* MD5  

## Instalation

```
yarn install
or
npm install
```

## How to use

```shell
node index.js --username edwinnnss
// or node index.js -u edwinnnss
User ID: 359695997

node index.js --id 359695997
// or node index.js -i 359695997
Username: edwinnnss
```
